想要學習資訊安全，卻不知道要怎麼開始？  
找不到適合自己的資訊安全學習素材？<br />
這份教學素材包將會用**簡單、容易**的方式來帶你學習「Prompt Injection」的攻擊手法！

## 總覽
這份教學素材包將會分成：
* 基本知識的介紹
* 不同的攻擊種類
* 駭客的攻擊應用
* 小試身手CTF題目
* 題目講解
透過這樣不同主題的分頁，來帶你認識 Prompt Injection 的知識！
